National Soil Monitoring Program Data Management

This tool is designed to allow soil site data entry into the NSMP database environment in a simple site data sheet format .

For more information about the National Soil Monitoring Program, please have a read at this website

The Process

  • You have opened this web tool with a sampling team token argument. While using the tool, only sites relevant to this token, will be available for processing. Update the token argument to access a different set of sites.
  • Download the MS Excel Site Data Entry Template from the "Data Ingestion" tab on this web site, for the sites related to your token.
  • Once you have downloaded copy the file, keep it somewhere safe for entering site data for this token. The MS Excel Site Data Entry workbook will be you point of truth for all the NSMP data you collect for these sites. We recommend storing it somewhere with access to cloud backup.
  • There will be one blank worksheet for every NSMP site you need to collect in the workbook.
  • Enter the data from your site sheet into the relevant site sheet in the workbook.
  • Do this for every site you wish to ingest into the database.
  • Where appropriate code lists will dop down for a given cell along with code descriptions.
  • When you have finished entering your data save the MS Excel file and drag the file into this the file entry box on the first tab (Morphology Data Ingestion) of the website. Or browse to the file to open.
  • Once upload is complete, you will then see a button "Validate Site Data". Click on this to do initial spreadsheet validation of all sites in your workbook. Sites in your workbook, that are already published, will be ignored.
  • If there are any data validation errors, these will be described so that you can fix them in the spreadsheet and reload.
  • Once there are no errors in the dataset you can click the "Import Site Data into holding DB" button to upload the data to the NSMP intermediate Database. This will test the data further and provide you with visualisation for you to review the data.
  • Review your site data in the various viewing tabs of this website to check you are happy with the data that has been uploaded. Ensure the data aligns data that has already been collected for the site. Check the location on the maps.
  • Once review is complete and the team is confident that the data is correct and valid, then publish these sites to the production database via the "Publish Sites" tab. Select the site(s) from the draft set, and click on the “Publish Selected Sites” button.
  • Once a site is published it can no longer be edited via this tool.
