National Soil Monitoring Program Data Management

This tool is designed to allow soil site data entry into the NSMP database environment in a simple site data sheet format .

For more information about the National Soil Monitoring Program, please have a read at this website

The Process

  • Download the MS Excel Site Data Entry Template from the "Data Ingestion" tab on this web site.
  • Once downloaded copy the file to somewhere safe for entering your data. This will be you point of truth for all the NSMP data you collect. We recomend storing it somewhere with access to cloud backup.
  • There will be one blank worksheet for every NSMP site you need to collect.
  • Enter the data from your site sheet into the relevant site sheet in the spreadsheet.
  • Do this for every site you wish to ingest into the database.
  • Where appropriate code lists will dop down for a given cell along with code descriptions.
  • When you have finished entering your data save the MS Excel file and drag the file onto this website.
  • You will then need to click the "Validate Site Data" button to run some checks on the data integrity.
  • Once there are no errors in the dataset you can click the "Import Site Data Into DB" button to upload the data to the NSMP Staging Database. Your site data will be ingested into a NatSoil schema DB and displayed on this website.
  • Once you have loaded your sites into the Staging Database, you can review them in the various viewing tabs of this website to check you are happy with the data that has been uploaded.
  • If you are happy with a site/s you can then publish these sites to the NSMP Site database via the "Publish Sites" tab.
  • Once a site is published it can no longer be edited via this tool.
